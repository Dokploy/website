---
title: 卸载
description: "了解如何在服务器上卸载 Dokploy"
---

import { Step, Steps } from 'fumadocs-ui/components/steps';

按照以下步骤，彻底从服务器中移除 Dokploy 及其组件。

<Steps>
<Step>
移除 Dokploy 创建的 Docker Swarm 服务：

   ```bash
   docker service remove dokploy dokploy-traefik dokploy-postgres dokploy-redis
   docker container remove -f dokploy-traefik
   ```
 
</Step>
 
<Step>
 
移除 Dokploy 创建的 Docker 卷（volumes）：

   ```bash
   docker volume remove -f dokploy-config dokploy-postgres dokploy-redis
   ```
 
</Step>

<Step>
 
移除 Dokploy 创建的 Docker 网络：

   ```bash
   docker network remove -f dokploy-network
   ```
 
</Step>
 
<Step>

执行 Docker 清理操作以删除残留资源：

   ```bash
   docker container prune --force
   docker image prune --all --force
   docker volume prune --all --force
   docker builder prune --all --force
   docker system prune --all --volumes --force
   ```
 
</Step>
 
<Step>

从服务器移除 Dokploy 文件和目录：

   ```bash
   sudo rm -rf /etc/dokploy
   ```
 
</Step>
</Steps>
